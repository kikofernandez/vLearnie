Coltrane Views
================

.. _coltrane_views:

Here you can take a look at the views that built the Coltrane application. The views in this section will have define a **Type** attribute that can be one of the followings:

* public, accessible to everyone
* private or internal, accessible only if you are editing your own space
* semi-public, accessible if the user has granted you permission
