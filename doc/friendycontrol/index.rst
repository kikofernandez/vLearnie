FriendyControl application
=============================
.. _friendycontrol_index: 

The purpose of this application is to control who can watch your space. We can customise the permission to let people see the files but not the blog and things like that.

The following sections explains how this application works:

* in :ref:`view <friendycontrol_views>` you can find the information related with this view,

* in :ref:`models <friendycontrol_models>` you can find how to create a specific object in this application.
